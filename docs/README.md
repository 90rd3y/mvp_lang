# Интерпретатор учебного языка программирования

Учебный интерпретатор строго типизированного языка программирования с русским синтаксисом. Написан на C++20 с нуля, без использования сторонних зависимостей.

## Ключевые архитектурные решения
* **Pipeline-архитектура:** Четкое разделение на `Lexer`, `Parser`, `Semantic` и `Interpreter`.
* **Data-Oriented Design (Flat AST):** Узлы синтаксического дерева хранятся в непрерывном `std::vector`. Связи между узлами реализованы через 32-битные индексы, что исключает использование умных указателей и повышает Cache Locality.
* **Bump Allocator (Arena):** Собственный менеджер памяти. Выделяет память чанками по 4 МБ. Используется для интернирования строк, типов и хранения рантайм-данных (массивов, структур). Гарантирует отсутствие утечек памяти (освобождается каскадно).
* **Pratt Parser:** Эффективный нисходящий разбор выражений с приоритетами операторов.
* **Memory Flattening:** Вложенные структуры сплющиваются в единый непрерывный блок памяти.

## Сборка
Требуется интерпретатор с поддержкой C++20 (GCC 10+ или Clang 10+).
```bash
make
```

## Использование
```bash
./myi <путь_к_файлу.lang>
```

Флаги отладки:
* `--dump-tokens` — вывести поток токенов лексера.
* `--dump-ast` — вывести плоское синтаксическое дерево.

## Тестирование
