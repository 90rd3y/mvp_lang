# Спецификация языка программирования

## 1. Лексика
* **Алфавит:** Поддержка UTF-8 (кириллица для ключевых слов и идентификаторов).
* **Комментарии:** Многострочные вложенные `/* ... */`.
* **Ключевые слова:** `целое`, `бцелое`, `вещ`, `лог`, `символ`, `строка`, `пусто`, `если`, `иначе`, `пока`, `прервать`, `продолжить`, `вернуть`, `истина`, `ложь`, `структура`, `тип`, `массив`, `функция`, `пространство`, `как`.
Примечание: типы также являются ключевыми словами.
Встроенные функции (`печать`, `ввод`, `выход`, `паника`, `утверждение`) теперь являются обычными идентификаторами, зарегистрированными в глобальной области видимости.

## 2. Система типов
Язык использует сильную статическую типизацию.
* **Базовые типы:** `целое` (int64), `вещ` (double), `лог` (bool), `строка` (string), `пусто` (void).
* **Составные типы:**
  * **Массивы:** Статические одномерные массивы `тип[размер] имя;`.
  * **Структуры:** Пользовательские типы `структура Имя { ... }`. Поддерживается вложенность.
  * **Синонимы:** `тип НовоеИмя = СтароеИмя;`.

## 3. Управление потоком
* **Ветвления:** `если (условие) { ... } иначе { ... }`
* **Циклы:** `пока (условие) { ... }`
* **Инструкции цикла:** `прервать;`, `продолжить;`
* **Функции:** Поддержка аргументов, возвращаемых значений и рекурсии. Точка входа — `функция main() : целое`.

## 4. Встроенные функции
(Смотри BUILTINS.md для подробностей)
* `печать(выражение)` — вывод в консоль.
* `ввод()` — чтение строки.
* `выход(код)` — завершение программы.
* `паника(сообщение)` — аварийное завершение.
* `утверждение(условие)` — проверка инвариантов (assert).

## 4.1 Операторы
* Арифметика: `+`, `-`, `*`, `/`, `%`
* Сравнения: `==`, `!=`, `<`, `>`, `<=`, `>=`
* Логика: `&&`, `||`, `!`
* Доступ: `.` (поле), `[]` (индекс), `::` (пространство имен)
* Приведение: `как<Тип>(выр)`

## 5. Грамматика (упрощенная EBNF)
```ebnf
Program       ::= NamespaceDecl | FuncDecl | StructDecl | TypeAlias | VarDecl | Stmt
NamespaceDecl ::= "пространство" Ident "{" Program* "}"
FuncDecl      ::= "функция" Ident "(" Params? ")" ":" Type (Block | ";")
StructDecl    ::= "структура" Ident "{" (Type Ident ";")* "}"
VarDecl       ::= Type ("[" Int "]")? Ident ("=" Expr)? ";"
Block         ::= "{" (VarDecl | Stmt)* "}"
Stmt          ::= IfStmt | WhileStmt | ReturnStmt | BreakStmt | ContinueStmt | ExprStmt | AssignStmt
AssignStmt    ::= Expr "=" Expr ";"
IfStmt        ::= "если" "(" Expr ")" Stmt ("иначе" Stmt)?
WhileStmt     ::= "пока" "(" Expr ")" Stmt
ExprStmt      ::= Call ";"
Expr          ::= Logic | Equality | Comparison | Term | Factor | Unary | Call | Index | Member | Literal | Cast
Cast          ::= "как" "<" Type ">" "(" Expr ")"
Call          ::= Ident ("(" Args? ")")? ("::" Ident)*
```
