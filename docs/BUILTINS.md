# Встроенные функции

В языке `MVP` реализованы следующие встроенные функции:

- `печать(значение)`: Выводит значение (целое, вещ, лог, строка) в консоль.
- `ввод()`: Читает строку из стандартного ввода (консоли). Возвращает тип `строка`.
- `выход(код)`: Завершает выполнение программы с указанным кодом возврата (тип `целое`).
- `паника(сообщение)`: Аварийно завершает выполнение программы, выводя сообщение (тип `строка`).
- `утверждение(условие)`: Проверяет инвариант (тип `лог`). Если условие ложно, программа падает.
